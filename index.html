Session 4 - Directive

What is Directive ?

  • Directives is used to attach custom behavior to elements in the DOM.
  • Directives can create, remove or manipulate existing DOM elements or its property.

Types of Directive.

  1. Attribute Directives.

    Change Attribute of an Existing DOM Element. <some-element [ngStyle]="{‘font-style’:’12px’}" >...</some-element>

  2. Structural directives.

    Update Existing DOM Element or Create New DOM Element. <p *ngIf="true"></p>

*ngIf

Include template, based on value.

Template: <p *ngIf="true/false/someFn()">...</p>

Add or Remove 'P' tag based on value.

*ngFor

Iterate Array and create elements based on length of the Array.

Variable: let Array = [ 1, 2, 3 ]

Angular Template: <p *ngFor="let i of Array">{{ i }}</p>

DOM Template: <p>1</p>
<p>2</p>
<p>3</p>

*ngFor

Variable: let Array = [ 'MRR', 'Artifi', 'Amla' ]

Angular Template: <p *ngFor="let i of Array">{{ i }}</p>

DOM Template: <p>MRR</p>
<p>Artifi</p>
<p>Amla</p>

[ngSwitch]

// Initialize Variable let name = 'MRR'
// Template Code <div [ngSwitch]="name">
     <div *ngSwitchCase="'MRR'"> name = MRR </div>
     <div *ngSwitchCase="'Artifi'"> name = Artifi </div>
     <div *ngSwitchCase="'Amla'"> name = Amla </div>
     <div *NgSwitchDefault> Default </div>
</div>

[ngClass]

// Template Code <div [ngClass]="{'active': isActive, 'disabled': isDisabled}">
// Output if isActive = true <div class="active" > </div>
// Output if isDisabled = true <div class="disabled" > </div>

Creating Custom Directive

Creating custom Directive to change the color of text

Generate new Directive using
ng g directive DIRECTIVE_NAME

CLI command will automatically import and declare directive into app.module.ts file.

Import ElementRef wrapper.

A wrapper around a native element inside of a View.

// bg-red.directive.ts import { Directive, ElementRef } from '@angular/core';

@Directive({
     selector: '[appBgRed]'
})

export class BgRedDirective {
     constructor(private el: ElementRef) {
          this.el.nativeElement.style.backgroundColor = 'red';
     }
}
